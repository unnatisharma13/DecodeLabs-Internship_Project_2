# Responsive Portfolio Website

## Project Overview

This project is a responsive portfolio website developed using HTML, CSS, and JavaScript. The website is designed using a mobile-first approach and adapts seamlessly across mobile, tablet, and desktop devices.

## Objectives

* Create a responsive webpage that works on different screen sizes.
* Implement CSS Grid and Flexbox layouts.
* Use media queries for responsiveness.
* Design a responsive navigation menu.
* Ensure proper spacing, alignment, and accessibility.

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Features

* Responsive Navigation Menu
* Mobile-First Design
* CSS Grid Layout
* Flexbox Layout
* Responsive Hero Section
* Skills Section
* Projects Section
* Contact Section
* Footer
* Smooth Scrolling

## Responsive Breakpoints

### Mobile

Default layout for screens below 768px.

### Tablet

Applied at min-width: 768px.

### Desktop

Applied at min-width: 1024px.

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open index.html in a web browser.
4. Alternatively, use the Live Server extension in VS Code.
